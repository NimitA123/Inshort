# Inshort
## About:-
This is News App.Where User can read top 100 news relative to the Entertainment, Sports, Science, Technology, Health etc.
## *Tech stack:-
Android SDK, Retrofit, Coroutine , LiveData and Android pattern(MVVM) ,Kotlin
